# notico

notico is Slack event notification bot.

## Usage

```
$ export SLACK_TOKEN=xxxxxx
$ notico [-channel "#foo"]
```

## Supported Events

- channel_created
- channel_deleted
- channel_rename
- channel_archive
- channel_unarchive
- team_join
- bot_added
